Goal: Age of the patient at the time of this visit, in years.

"value" must be a float (e.g. 6.0, 14.5). This field is required — you must always produce a value.

RULES OF PRECEDENCE:
- If age is explicitly stated in the narrative text for THIS visit (e.g., "6-year-old girl"), this takes highest priority.
- If age is not restated in the narrative text, use structured age information combined with visit timing to compute age.
- Do NOT simply repeat the same age across visits if sufficient information exists to compute an updated age.

Computation rules:
- If a baseline age (0-month visit) is known and this visit is a follow-up at approximately 6 months, add 0.5 years.
- If this visit is approximately 12 months after baseline, add 1.0 year.
- If DOB and visit date are both present, compute age directly to one decimal place.
- If age is described approximately (e.g., "teenage boy"), approximate and explain in reasoning.
- If age is completely absent but pediatric context is clear (school-age child, infant, etc.), use your best estimate and set confidence to "low".
- You must always output a float. There is no unknown value for this field.

IMPORTANT:

3. OnsetAgeYears

Goal: Age of the patient in years when seizures first began.

"value" must be a float (e.g. 0.75, 3.0, 12.5) or -1 if unknown.

STRICT RULES:

- If the note states onset age directly (e.g. "seizures started at age 3", "onset at 2 years"), use that value.
- If the note states onset in months (e.g. "first seizure at 9 months"), convert to years (e.g. 0.75).
- If the note states "since birth" or "neonatal onset", use 0.0.
- If the note states "since infancy" without more detail, use 0.5 as an approximation and set confidence to "low".
- Do NOT compute or infer onset age from current age minus duration. Only extract what is explicitly stated or directly implied.
- If onset age is not described in this note, set "value" to -1.

4. SeizureFreq

Goal: Classify seizure burden over the last 12 months into one of 5 strict categories.

"value" must be exactly one of:
- 0 (Seizure Free): Absolutely no seizures for at least 1 year.
- 1 (Infrequent): Very rare events, occurring less than once a month (e.g., once or twice a year).
- 2 (Monthly): Occurring roughly once a month, but less than weekly.
- 3 (Weekly): Occurring at least once a week, or monthly events that happen in big clusters.
- 4 (Daily): Very severe burden, occurring every day or multiple times a day.
- -1 if seizure frequency is not mentioned at all in this note.

Map the text to the closest class. Use the time frame implied by the note. If the note only gives a recent isolated event (e.g. "2 days ago") without a rate, use clinical judgment to assign the best class or -1 if truly unclassifiable.

5. CognitivePriority

"value" must be exactly one of:

- 2: Severe concerns — severe intellectual disability, profound developmental delay, developmental encephalopathy, cerebral palsy with major functional impairment, or any condition explicitly described as severely impairing.

This field is required — you must always produce 0, 1, or 2. If there is no mention of cognitive or behavioral concerns, output 0.

6. SeizureType

Goal: Dominant seizure type described in this note.

- 2 (Generalized Motor): Generalized seizures with prominent motor features — generalized tonic-clonic, generalized convulsions, whole-body shaking, tonic seizures.
- -1: Seizure type not clearly described in this note, or no seizures mentioned.

If the note describes mixed or unclear semiology, assign the most prominent type. If truly ambiguous, use -1.
